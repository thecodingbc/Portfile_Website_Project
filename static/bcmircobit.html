MICROBIT DESCRIPTION


/

A micro:bit is a pocket-sized computer that introduces block coding coders how to let software and hardware work together. It has an LED light display, buttons, sensors and many input/output features that, when programmed, let it interact with the coder and his/her world.
Posted on March 30, 2017 by tballmsft. Microsoft MakeCode is a web-based environment for learning to code with physical computing devices such as the micro:bit. It has a handful of novel features all packed into a single web application (web app).
